Télécharger en PNG Imprimer / Créer PDF

Pour créer un PDF à partir de cette image :

  1. Cliquez sur le bouton "Imprimer / Créer PDF"
  2. Dans la boîte de dialogue d'impression, sélectionnez "Enregistrer comme PDF" ou "Microsoft Print to PDF"
  3. Choisissez un emplacement pour enregistrer votre fichier PDF
  4. Cliquez sur "Enregistrer" ou "Imprimer"

Note : Cette méthode fonctionne sur la plupart des navigateurs modernes et ne nécessite aucun logiciel supplémentaire.
